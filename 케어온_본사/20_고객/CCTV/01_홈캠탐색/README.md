# [[01_홈캠탐색]]

## 1. 고객 특징

- **니즈**: 가정용 홈캠 찾기
- **목적**: 집 안 모니터링, 반려동물 관찰, 아이 케어 등
- **특성**: 개인 고객, 간단한 설치 선호

---

## 2. 페인 포인트

- 어떤 제품이 좋은지 모름
- 설치가 어려울 것 같음
- 가격 대비 성능 비교 어려움

---

## 3. 케어온 솔루션

- 쉬운 설치 가이드 제공
- 렌탈로 부담 없이 시작
- 성능 좋은 제품 추천
