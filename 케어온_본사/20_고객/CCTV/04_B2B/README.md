# [[04_B2B]]

## 1. 고객 특징

- **범위**: CCTV 설치가 필요한 모든 사업자
  - 신규 창업자
  - 기존 자영업자
  - 빌라 관리자
  - 상가 건물주
  - 병원, 의원
  - 기타 B2B 고객 전체

- **특성**: 
  - 사업 운영에 CCTV 필수
  - 비용 절감 중요
  - 안정적인 시스템 필요

---

## 2. 페인 포인트

- 초기 설치 비용 부담
- 유지보수 걱정
- 업체 선택의 어려움
- 계약 조건 복잡함

---

## 3. 케어온 솔루션

- 렌탈로 초기 비용 부담 최소화
- 정기 유지보수 포함
- 위약금 없는 자유로운 계약
- 맞춤형 패키지 구성

---

## 4. 향후 세분화 계획

> **현재는 3-2 고객에 집중하므로 아직 세분화하지 않음**

향후 세분화 예정 카테고리:
- 신규 창업자
- 기존 자영업자
- 빌라/아파트
- 상가 건물
- 병원/의원
- 기타 업종별
