## 🎯 기록 원칙

### 1. 날짜 기반 네이밍
```
[[기록_YYMMDD_제목]]
예: [[기록_251118_신예준통화]]
```

- 통화 직후 전사
- 아이디어 즉시 메모

- 99_기록은 임시 보관소
- 정기적으로 분류 및 정리
