# 지식 (Knowledge Wiki)

> **케어온의 나무위키 / 위키피디아**

---

## 📌 역할

이 폴더는 **지식 베이스(Knowledge Base)**입니다:
- 개념 정의 (창조적 파괴, 천민자본주의 등)
- 용어 설명 (눈탱이, 자영업자 범위 등)
- 방법론 (효율화, 자동화 전략 등)
- 사례 연구
- 철학 및 가치관

직원이나 AI가 작업 중 모르는 개념이 나오면 → 이 폴더의 문서로 이동 → 이해 후 복귀

---

## 🏷️ 태그 시스템

모든 지식 문서는 **태그로 분류**합니다.

### 주요 태그
- `#비전` - 비전 관련 개념
- `#미션` - 미션 관련 개념
- `#전략` - 전략 및 방법론
- `#고객` - 고객 관련 용어
- `#제품` - 제품/서비스 관련
- `#마케팅` - 마케팅 용어
- `#철학` - 철학적 개념
- `#경제학` - 경제 이론
- `#용어` - 일반 용어 정의
- `#개념` - 추상적 개념
- `#방법론` - How-to 지식

### 태그 검색 예시
```
옵시디언에서:
- tag:#비전 → 비전 관련 모든 문서
- tag:#고객 → 고객 관련 모든 문서
- tag:#용어 → 용어 정의들
```

## 📝 문서 작성 규칙

### 파일명
```
창조적파괴.md        (O) - 단순하게
개념_창조적파괴.md    (X) - 접두사 불필요
```

### 메타데이터 (YAML Frontmatter)
```yaml
---
분류: 지식
태그: [비전, 개념, 경제학]
관련문서: [[천민자본주의]], [[혁신]]
설명: 한 줄 요약
작성일: 2025-MM-DD
---
```

### 문서 구조
```markdown
# 제목

## 정의
명확한 정의 1-2문장

## 설명
상세 설명

## 예시
구체적 사례

## 케어온에서의 적용
우리 비즈니스와의 연결

## 관련 지식
- [[관련문서1]]
- [[관련문서2]]
```

## 🔗 링크 규칙

## 🎯 사용 시나리오

### 시나리오 1: 직원 온보딩
신입: "창조적 파괴가 뭐죠?"  
→ `지식/창조적파괴.md` 읽기  
→ 이해 후 업무 복귀

### 시나리오 2: AI 작업
AI: "[[눈탱이]]의 정의가 필요합니다"  
→ `지식/눈탱이.md` 참조  
→ 맥락 파악 후 작업 진행

### 시나리오 3: 문서 작성
마케터: "블로그에 '천민 자본주의' 언급하려는데..."  
→ `지식/천민자본주의.md` 확인  
→ 정확한 정의 사용
