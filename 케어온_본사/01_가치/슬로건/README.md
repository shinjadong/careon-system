# 슬로건

> **한 문장으로 압축한 우리의 정체성**

---

## 📋 슬로건 문서

### 내부 (직원용)
**"막다른 길, 그곳이 고속도로다"**

- 의미: 모두가 불가능하다고 할 때, 그곳이 기회다
- 레드오션에서 틈새 찾기
- 거인의 등에 올라타기

### 외부 (고객용)
1. **"위약금 없는 렌탈, 케어온"**
   - 주력 메시지
   - 차별화 포인트

2. **"사장님의 모든 것, 케어온"**
   - 종합 솔루션 제공

3. **"신뢰의 다른 이름, 케어온"**
   - 브랜드 가치

---

## 🎯 슬로건의 역할

- **내부**: 팀원의 사고방식 통일
- **외부**: 고객에게 전달할 핵심 메시지
- **브랜딩**: 일관된 메시지 전달
