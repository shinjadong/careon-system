**그렇게 해야 하는 이유**:
```
- 케어온은 '사업'이다.
- 사업의 제 일 목표는 '이윤 추구'이다.
- 사업체는, 창업자와 핵심구성원의 적극적 기여를 연료로 사용한다.
    - 창업자는 인간이다.
    - 인간은 보상 없는 기여를 지속할 수 없다.
    - 바꿔 말해, 보상을 극대화한다면 기여를 극대화할 수 있다.
```
