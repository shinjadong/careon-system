- 동기화 규칙 (GitHub-Obsidian-Notion)
- 문서 작성 가이드

---

## 🎯 핵심 원칙

### 1. 직관성
**"엑셀 헤더처럼 제목만 보고 내용을 알 수 있어야 한다"**
- 모호한 제목 금지
- `[[폴더명_핵심키워드]]` 형식 준수

### 2. 구조화
**"모든 문서는 정해진 폴더 중 하나에 속해야 한다"**
- 고아 문서(Orphan) 없음
- 상위문서 명시

### 3. 연결성
**"모든 문서는 상위 목적과 연결되어야 한다"**
- 비전/미션과 무관한 문서 없음
- 양방향 링크 구축
